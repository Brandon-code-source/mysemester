# MySemester

AI-powered academic operating system for students.

## Features

- Upload syllabus PDFs
- AI extracts assignments, quizzes, exams, and deadlines
- AI Study Coach with contextual academic guidance
- PostgreSQL cloud database
- Fullstack architecture using Next.js + FastAPI
- Persistent course dashboard

## Tech Stack

### Frontend
- Next.js
- React
- TailwindCSS

### Backend
- FastAPI
- OpenAI API
- SQLAlchemy

### Database
- PostgreSQL (Supabase)

## Vision

## Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
